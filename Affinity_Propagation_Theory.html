Parâmetros do 'AffinityPropagation'¶

dampingfloat (padrão=0,5): O fator de amortecimento no intervalo [0,5 ; 1,0] é a extensão em que o valor¶

atual é mantido em relação aos valores de entrada (ponderação 1 - amortecimento).¶

Isso é feito para evitar oscilações numéricas ao atualizar esses valores (mensagens).¶

max_iterint (padrão=200): Número máximo de iterações.¶

convergence_iterint (padrão=15): Número de iterações sem alteração no número de clusters estimados que¶

interrompe a convergência.¶

copybool (padrão=True): Fazer uma cópia dos dados de entrada.¶

preferencearray-like de formato (n_samples,) ou float (padrão=None): Preferências para cada ponto - pontos¶

com valores maiores de preferências têm maior probabilidade de serem escolhidos como exemplares.¶

O número de exemplares, ou seja, de clusters, é influenciado pelo valor das preferências de entrada.¶

Se as preferências não forem passadas como argumentos, elas serão definidas como a mediana das similaridades¶

de entrada.¶

affinity{‘euclidean’, ‘precomputed’} (default=’euclidean’): Qual afinidade usar. No momento, ‘precomputed’¶

e euclidean são suportados. ‘euclidean’ usa a distância euclidiana ao quadrado negativo entre os pontos.¶

verbosebool (default=False)¶

random_stateint, instância¶

RandomState ou None (default=None): Gerador de números pseudoaleatórios para controlar o estado inicial.¶

Use um int para resultados reproduzíveis entre chamadas de função.¶

Agrupamento po Afinidade de Propagação com DADOS ALEATÓRIOS¶

A função make_blobs do sklearn.datasets gera um conjunto de pontos n-dimensional, em que cada ponto¶

pertence a um dos k clusters.¶

A classe AffinityPropagation do sklearn.cluster faz o agrupamento nos dados.¶

Definimos o parâmetro preference como -50, que controla o número de clusters a serem gerados.¶

Valores menores de 'preference' resulta em mais clusters sendo gerados. Depois avaliamos a qualidade do agrupamento.¶

from sklearn.datasets import make_blobs

df4, labels_true = make_blobs(n_samples=500, cluster_std = 1, random_state = 0)

af = AffinityPropagation(preference = -30 , random_state = 0).fit(df4)
cluster_centers_indices = af.cluster_centers_indices_
labels = af.labels_

n_clusters_ = len(cluster_centers_indices)

print("Número estimado de clusters: %d" % n_clusters_)
print("Coeficiente de Silhueta: %0.3f" % mt.silhouette_score(df4, labels, metric="sqeuclidean"))

Preference = [-50, -40, -30, -20, -10 , 0]
silhouette_scores = []

for preference in Preference:
    model = AffinityPropagation(preference=preference, random_state=42)
    model.fit(X_std)
    
    # Evaluate only if more than one cluster is found
    if len(np.unique(model.labels_)) > 1:
        score = mt.silhouette_score(X_std, model.labels_)
        silhouette_scores.append(score)
    else:
        silhouette_scores.append(np.nan)

plt.plot(Preference, silhouette_scores, marker='o')
plt.title('Preference vs Silhouette Score')
plt.xlabel('Preference')
plt.ylabel('Silhouette Score')
plt.grid()
plt.show()
